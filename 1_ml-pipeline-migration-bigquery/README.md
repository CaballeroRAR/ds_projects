# ML Pipeline Migration to BigQuery

## Overview
This project focuses on migrating an existing Machine Learning pipeline to Google Cloud Platform's BigQuery. It leverages BigQuery ML for scalable model training and evaluation directly within the data warehouse.

## Project Goal
- Transition from local/server-based processing to cloud-native data warehousing.
- Implement ETL processes for BigQuery.
- Optimize SQL queries for ML feature engineering.
